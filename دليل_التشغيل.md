# دليل تشغيل صفحة الهبوط - بنشر متنقل الكويت

## 📋 متطلبات التشغيل

- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn
- Git (اختياري)

## 🔧 التشغيل المحلي (Local Development)

### 1. تحميل المشروع
```bash
# إذا كان لديك رابط المشروع من GitHub
git clone [رابط_المشروع]
cd [اسم_المشروع]

# أو قم بتحميل ملفات المشروع مباشرة
```

### 2. تثبيت المكتبات المطلوبة
```bash
npm install
```

### 3. تشغيل السيرفر المحلي
```bash
npm run dev
```

سيتم تشغيل الموقع على: `http://localhost:5173`

### 4. بناء النسخة النهائية للنشر
```bash
npm run build
```

## 🌐 النشر على الإنترنت

### أ) GitHub Pages (مجاني 100%)

**المزايا:**
- مجاني تماماً
- سهل التحديث
- رابط مجاني: `اسمك.github.io/اسم-المشروع`

**الخطوات:**
1. ارفع المشروع على GitHub
2. اذهب إلى Settings > Pages
3. اختر Source: GitHub Actions
4. سيتم إنشاء رابط مجاني تلقائياً

**ملاحظة:** GitHub Pages يدعم المواقع الثابتة (Static) وهذا المشروع مناسب له تماماً!

### ب) Netlify (مجاني + خطط مدفوعة)

**المزايا:**
- نشر مجاني مع ميزات متقدمة
- دومين مجاني (.netlify.app)
- ربط دومين مخصص مجاناً
- تحديث تلقائي من GitHub

**الخطوات:**
1. سجل في [netlify.com](https://netlify.com)
2. اربط حساب GitHub
3. اختر المشروع واتركه ينشر تلقائياً

### ج) Vercel (مجاني + خطط مدفوعة)

**المزايا:**
- أداء عالي جداً
- دومين مجاني (.vercel.app)
- ربط دومين مخصص مجاناً
- تحديث فوري

**الخطوات:**
1. سجل في [vercel.com](https://vercel.com)
2. اربط GitHub واختر المشروع
3. سيتم النشر تلقائياً في ثوانٍ

## 🏆 أفضل خيارات الاستضافة

### للمبتدئين (مجاني):
1. **GitHub Pages** - الأسهل والأكثر موثوقية
2. **Netlify** - ميزات أكثر وسهولة في الإدارة
3. **Vercel** - الأسرع في التحميل

### للاستخدام التجاري:
1. **Cloudflare Pages** - سريع جداً + حماية قوية
2. **AWS S3 + CloudFront** - للمواقع الكبيرة
3. **استضافة محلية** مثل:
   - استضافة السعودية (SaudiCloud)
   - استضافة الكويت (Kuwait Host)

## 💡 نصائح مهمة

### ربط دومين مخصص:
```
بدلاً من: https://yoursite.netlify.app
استخدم: https://bansharkuwait.com
```

**الخطوات:**
1. اشترِ دومين من (Namecheap, GoDaddy, etc.)
2. في إعدادات الاستضافة، أضف الدومين المخصص
3. عدّل DNS Records كما هو مطلوب

### تحسين الأداء:
```bash
# ضغط الصور قبل الرفع
npm install -g imagemin-cli
imagemin src/assets/* --out-dir=dist/assets
```

### SSL مجاني:
جميع المنصات المذكورة تقدم شهادات SSL مجانية تلقائياً

## ⚡ تشغيل سريع (خطوة واحدة)

### GitHub Pages:
1. ارفع الملفات على GitHub
2. فعّل GitHub Pages
3. ✅ جاهز!

### Netlify Drag & Drop:
1. اعمل `npm run build`
2. اسحب مجلد `dist` إلى netlify.com
3. ✅ جاهز!
